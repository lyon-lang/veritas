CoreValidate Extension Test Page

How to Test

  1. Install the CoreValidate extension in your browser
  2. Load this page
  3. Look for the trust score badge in the bottom-right corner
  4. Hover over the badge to see the detailed breakdown
  5. Click the badge to open the dashboard

Extension Features

  • Trust Score Badge: Automatically appears on every page
  • Hover Tooltip: Shows detailed breakdown of checks
  • Click to Open: Opens the full report in the dashboard
  • Dismiss: Click the × to hide the badge

Browser-Specific Notes

  • Chrome: Full support, service worker background
  • Firefox: Full support, may need temporary install
  • Edge: Full support, Chromium-based
  • Safari: Requires Xcode build, Safari 17+
